# WiX toolset 4 CLI
### Downloads and documentation
<b>Download links:</b> <br />
* [SumatraPDF](https://www.sumatrapdfreader.org/download-free-pdf-viewer)

<b>Generate package guid</b>
```powershell
[guid]::NewGuid().guid
```

<b>WiX build MSI package command</b>
```powershell
wix build SumatraPDF.wxs
```
