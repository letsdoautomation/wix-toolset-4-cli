### Downloads and documentation
<b>Download links:</b> <br />
* [K-Lite Code Pack](https://www.codecguide.com/download_kl.htm)

<b>Silent switches:</b>
```powershell
/VERYSILENT /NORESTART
```

<b>Generate package guid</b>
```powershell
[guid]::NewGuid().guid
```

<b>WiX build MSI package command</b>
```powershell
wix build "klite.wxs"
```
